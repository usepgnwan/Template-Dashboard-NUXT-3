create new project : 
```
npx nuxi@3.11.1 init nuxt3-belajar
```
create layout : 
```
 npx nuxi add layout default
```
create page :
```
npx nuxi add page index
```  

install nprogress :
```
npm install nprogress
```
i scroll :
```
npm install tailwind-scrollbar
```
